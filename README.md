## 说明

按尽可能延伸相邻书架的规则划分书架，三层中文阅览区共计 **133** 个书架。

- 划分规则：沿书架较长边所在直线方向延伸的，不含较大间隔（如座位和较宽过道）的最长连通块。
- 书架排序：近似按书架最小索书号的**分类号**为第一关键字，**索书号**为第二关键字升序排列。
